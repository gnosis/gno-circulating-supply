# GNO Circulating Supply

## Install
```sh
npm i
# or
yarn
```

## Run
```sh
node index.js

```

### Environment variables (need setup)
* **DEBUG="*"** it will output all debug statements.
* **NODE_URL** complete URL for JSON RPC Ethereum API (including port and apikey)
* **BUCKET_NAME** AWS S3 bucket name
* **AWS_ID**
* **AWS_SECRET**
